PLTL has 3 main User Roles:

  • Instructor
  • Peer Leader
  • Student

    How to create Course?

    Admin of the site can only create Course.

     
    How to create Class?

    Only Instructor can create Class: In the instructors home page, Under the Create Class tab: Add Class ID, Select Course ID from drop down list, Select Semester from drop down list, Add year, Add Class Description (Optional) and hit Create New button.

     
    How to view Instructors current Courses?

    In the instructors home page, Under Create Class tab there is a data table (Current Classes) with the column names: Class ID, Course Name, Semester and Year. Instructor will be able to see Classes and Courses his teaching in current semester.

     
    How to Search for Class (Instructor View)?

    In the instructors home page, Under Search Class tab insert Class ID and hit Search button. If record doesnt exist there will be a massage pop up 'No Record Found. Please type full Class ID'. If record exist you can view in the data table(Search Result) below.

     
    How to update Class details (Instructor View)?

    In the search result table there will be a button 'Edit'. Hit the button there will be a modal pop up containing details of the Class. You will only be able to update Semester, Year and Class Description of the Specific Class you searched for.

     
    How to view Peer Leader/Student current registered Classes?

    In the Peer Leader/Student home page, Under Search Class tab there is a data table (View Class) in the top right side with the column names: Class ID, Status.

     
    How to Search for Class (Peer Leader/Student View)?

    In the Peer Leader/Student home page, Under Search Class tab insert Class ID and hit Search button. If record doesnt exist there will be a massage pop up 'No Record Found. Please type full Class ID'. If record exist you can view in the data table below.

     
    How to register/drop Class (Peer Leader/Student View)?

    In the Peer Leader/Student home page, Under Search Class tab insert Class ID and hit Search button and there will be a data table (Search Result) and a Description button. Hit description button and from the modal you can view description of the class and Register for the class by hitting Register button. Once you register you will be able to see update data table (View Class) in top right of the page. Once you registered you can always drop class. For dropping specific class hit Drop button of the Class ID.

     
    How To View List of Assignments (Instructor View)?

    In the Instrucotors home page, Under Create Class tab instructor can view all the current classes his teaching for the semester or search for specific class under Search Class tab. From data table click Class ID link and will redirect to Assignment home page for the particular Class ID. In the Assigment list table you can view All the Assignments, Due date and Grade.

     
    How To Upload New Assignment (Instructor View)?

    In the Assigment home page, Hit Upload Assignment button you will be redirected to a new page where you can upload assignment. Data field for uploading assigment: Assignment name, Publish date, Due date, Total grade of the particular Assigment and Upload file option. If you want to cancel and return back to previous page hit Cancel.

     
    How To Manage Student and Peer Leader Group (Instructor View)?

    In the Assigment home page, Hit upload Manage button you will be redirected to a new page where you can manage all the student and peer leader who has been registered for specific class. Instructor will be able to view all the registered student/peer leader and Enroll them in his class.

     
    How To Assign Peer Leader (Instructor View)?

    In the Manage Student home page, Instructor can view all the registered student and after enrolling them he can assign peer leader for specific student group from the user list drop down menu.

     
    How To Upload Home Work Solution (Student View)?

    In the Home page hit search for class and choose Class ID from search table. You will be redirected to Assigment page where you will be able to see Upload date, Due date, Total grade. After uploading solution for specific assigment student can view/download uploaded solution.

     
    How To View All Grades?
  • Instructor can view all the assigments in a data table and grade particular class.
  • Peer Leader can view all the assigments grade of his student group only.
  • Student can only view his own grade for particular class.
    •  
    How to view and grade assignment (Instructor View)?

    In instructors class page, from assigment list click on assignment name you will be redirected to view grade page. Instructor can open assignment, download assignment and grade assignment.
